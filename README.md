Cloud Development POE – Artisan Craftwork Web Application
📌 Project Overview

This project is a cloud-based web application developed to support artisans by providing a platform to showcase and sell their craftwork globally. The system allows customers from around the world to browse, view, and order handcrafted products online.
The application focuses on promoting local craftsmanship while solving the problem of limited market access for artisans.

Problem Statement

Many talented artisans struggle to reach a global audience due to:
Limited online presence
Lack of digital platforms to showcase their work
Difficulty managing orders and customer interactions
This web application addresses these challenges by providing a centralized platform where artisans can display their work and customers can easily access and purchase products.

Cloud Computing & Relevance

Cloud computing plays a key role in this project by:
Hosting the web application online using Microsoft Azure
Allowing global accessibility of the platform
Providing scalability to handle multiple users and orders
Ensuring secure data storage using cloud databases
This demonstrates how cloud technology can be used to build modern, scalable, and accessible business solutions.

Technologies & Tools Used

The following tools and platforms were used to develop this project:
Visual Studio Code – Code editor for development
Microsoft SQL Server – Database management system
Microsoft Azure – Cloud platform for hosting and deployment

Application Features

Home Page:
Displays information about the platform
Showcases the craftsmanship and artwork created by artisans
Provides users with an overview of available products

About Us
Provides detailed information about James Kumalo
Explains his background and role in the craftwork business
Highlights the purpose and vision of the platform

Contact Us
Displays contact details for Osmond’s Artwork
Allows customers to reach out for inquiries or support

Craftwork & Pricing
Displays the different types of craftwork created by James Kumalo
Includes pricing information for each item
Enables customers to view and select products for purchase

Deployment

The application is deployed using Microsoft Azure, making it accessible globally via the internet.
This ensures:
High availability
Scalability
Reliable performance

System Functionality

The system allows:
Storage of artisan craftwork in a database
Display of products to users worldwide
Customers to browse and place orders
Easy management of product information and pricing
